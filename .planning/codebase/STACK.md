# Technology Stack

## Frontend
- **HTML5**: Semantic structure.
- **CSS3**: Vanilla CSS with modern properties (Flexbox, Grid, Inter font).
- **JavaScript (ES6+)**: Vanilla JS for interactivity.
- **Vite**: Likely used for development (based on port 5173).
- **Intersection Observer API**: Used for scroll animations.
- **Google Fonts**: Inter.

## Backend
- **Node.js**: Runtime.
- **Express**: Web framework.
- **CORS**: For cross-origin requests from frontend.
- **Dotenv**: For environment variable management.

## Deployment/Infrastructure
- **Localhost**: Currently running on `localhost:5173` (frontend) and `localhost:5000` (backend).
