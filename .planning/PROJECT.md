# Cliq Campus

## What This Is
Cliq Campus is a modern platform designed to simplify the global education journey for students, partners, and universities. It provides tools for scholarship discovery, financial aid planning (EMI calculator), and preparation class coordination.

## Core Value
Simplifying the path to international education through AI-driven scholarship matching and transparent financial planning.

- ✓ Sticky navigation bar with mobile menu — Phase 0
- ✓ Scholarship application form (Frontend + Backend integration) — Phase 0
- ✓ EMI Calculator for financial aid — Phase 0
- ✓ FAQ section — Phase 0

## Context
- **Technical Environment**: Monorepo with static frontend and Express backend.
- **User Research**: Targeting students looking for global scholarships, particularly from Nepal/India.

## Constraints
- **Tech Stack**: Vanilla HTML/CSS/JS (Frontend), Node.js/Express (Backend).
- **Environment**: Local development on port 5173 (Frontend) and 5000 (Backend).

## Key Decisions
| Decision | Rationale | Outcome |
|----------|-----------|---------|
| Vanilla JS/CSS | Maximum flexibility and no build overhead for simple landing page. | ✓ Good |
| Decoupled Backend | Separation of concerns and scalability. | ✓ Good |
